# Daily Life Quest Catalog (50, Quest-Level)

아래 목록은 **미션 50개가 아니라 퀘스트 50개**다.
각 항목은 상위 목표(퀘스트)이며, 내부에 4~6개 미션으로 분해되는 것을 전제로 작성했다.

## 기준
- 총 50개 퀘스트 (10개 카테고리 x 5개)
- 각 퀘스트는 단일 행동이 아닌 **묶음 목표**
- 컬럼 `대표 미션 묶음`은 실제 분해 시 사용할 하위 미션 흐름 예시
