- [x] `캐릭터 상태`는 `홈/스탯` 탭에서만 노출
- [x] `태스크 청킹`은 `홈` 탭에서만 노출
- [x] `스탯` 탭에서도 캐릭터 상태 카드 노출 (요청 반영)
- [x] 대기중 퀘스트 아이콘/우측 화살표 UI를 `main_ui` 기준으로 보정
- [x] 대기중 퀘스트 메타(`시작(Start)/마감(Due)/소요(Est. Min.)`)를 compact 3칩으로 복구
- [x] 아코디언 아이콘을 우상단 absolute 배치로 재수정(본문 레이아웃 영향 제거)
- [x] 대기중 메타 셀 카드 배경/보더 제거, 괄호 표기 제거, 아이콘/패딩/간격 타이트 조정
- [x] 대기중 메타를 괄호 포함 표기(`(Start)/(Due)/(Est. Min.)`) + 좌측정렬로 재조정
- [x] 몬스터 아이콘/텍스트 거리 및 카드 패딩/줄간격을 `main_ui` 비율에 맞게 추가 타이트닝
- [x] 대기중 퀘스트 외부 래퍼 카드 제거 + 타이틀 텍스트 분리, 행 간격/줄간격 가독성 보정
- [x] `퀘스트 추가/편집` 모달을 버튼형 시간 카드(시작/마감/소요) + 하단 단일 CTA 구조로 보정
- [x] 소요시간 프리셋을 `-5분/-1분/+1분/+5분`으로 교체하고, 소요시간 변경 시 `마감=시작+소요` 강제 동기화
- [x] 태스크 입력 섹션 하단 설명 텍스트 제거 + 상단 버튼 라벨 `퀘스트 추가`로 정리
- [ ] 390/430/768 반응형 시각 정합 수동 점검
- [ ] 접근성(포커스/대비/터치 타깃) 수동 점검

## 1) 상단 타이틀 바 (`/docs/main_ui.png`)
- 좌측: `ADHDTime` + `레벨 12 (LV.12) 모험가` + 소형 아바타
- 우측: `오늘의 달성도 DAILY PROGRESS! 75%` + 원형 프로그레스/아바타
- 요구 정합:
- [ ] 좌우 정보 블록의 수직 기준선 정렬
- [ ] 좌측 브랜드 블록(타이틀/레벨) 타이포 계층 유지
- [ ] 우측 75% 시각 강조(링/텍스트 대비) 유지

## 2) 캐릭터 상태 (`/docs/main_ui.png`)
- 섹션 타이틀: `캐릭터 상태`
- 카드: 좌측 캐릭터 일러스트, 우측 레이더 5축
- 축 라벨: `시작력/몰입력/분해력/회복력/지속력` + 영문 보조 라벨
- 요구 정합:
- [ ] 카드 라운드/그림자/패딩이 기준 이미지와 유사
- [ ] 레이더 폴리곤 색상 레이어와 축 라벨 위치 재현
- [ ] 캐릭터 일러스트와 차트 사이 간격 균형 유지

## 3) 태스크 청킹 (`/docs/ui.png`)
- 섹션 타이틀: `무지성 태스크 청킹 (AI TASK CHUNKING)`
- 입력영역: 검색형 필드 + 마이크 액션
- CTA: `AI가 쪼개기 (CHUNK!)`
- 보조 디테일: 엄지 접근 가이드 콜아웃(`Easy for thumb access`)
- 요구 정합:
- [ ] 입력창/마이크/CTA가 한 손 조작 영역에서 2탭 이내 동선
- [ ] 한국어+영문 병기 라벨의 줄바꿈/오버플로우 안정화
- [ ] 입력 placeholder와 보조 아이콘 대비 확보

## 4) 현재 퀘스트 (`/docs/main_ui.png`)
- 헤더: `현재의 퀘스트`, 우측 `1개 +`
- 확장 카드: `마감 전! 설거지`, 대형 타이머 `08:14`, 몬스터 일러스트
- CTA 3종: `시작`, `일시정지`, `완료!`
- 하단 보조 문구: `5분마다 미세 진동 알림`
- 요구 정합:
- [ ] 타이머가 카드의 1순위 시각 계층으로 인지
- [ ] 3개 CTA의 색상/크기/간격/라운드 일치
- [ ] 보조 알림 텍스트가 과도하게 강조되지 않도록 보조 계층 유지

## 5) 대기중 퀘스트 (`/docs/main_ui.png`)
- 섹션 타이틀: `대기 중인 퀘스트`
- 리스트: 좌측 몬스터 아이콘 + 퀘스트명 + 시작/마감/예상 소요 메타 정보
- 우측 꺾쇠: 행 확장/접기 트리거
- 요구 정합:
- [ ] 행 높이/아이콘 크기/텍스트 간격을 압축형 리스트 톤으로 유지
- [ ] 메타 정보(`시작/마감/소요`) 가독성 유지
- [ ] 체크/확장 인터랙션의 충돌(이벤트 버블링) 방지

## 6) 퀘스트 추가/편집 (`/docs/main_ui.png`)
- 모달 타이틀: `퀘스트 추가/편집` + `X` 닫기
- 입력: `퀘스트 이름` + 값 `청소하기`
- 시간 카드 3종: `시작 예정`, `마감 기한`, `소요 시간`(버튼 클릭 선택/입력)
- 프리셋 칩: `-5분`, `-1분`, `+1분`, `+5분`
- 하단 CTA: `퀘스트 생성!`
- 요구 정합:
- [x] 하단 시트형 오버레이 계층과 배경 분리
- [x] 입력 -> 시간카드 -> 프리셋 -> 생성 CTA 순서 고정
- [x] 소요시간 변경 시 시작시간 대비 마감시간 자동 동기화
- [ ] 닫기/생성 후 포커스 복귀와 스크롤 잠금 해제 정상화

## 7) 나머지 세부 요소 (`/docs/ui.png`)
- 하단 탭바 아이콘 스타일/영문-한글 병기 톤
- 상태 라벨 보조 텍스트(예: `ADAPTIVE TIMER`, `Haptic Guide`)
- 보조 화살표/콜아웃 등 설명성 장식 요소
- 요구 정합:
- [ ] 탭 활성 상태(`홈(HOME)`) 색상 대비와 아이콘 강조 유지
- [ ] 혼합 언어 라벨에서 폰트 크기/행간 불일치 최소화
- [ ] 설명성 장식 요소는 본문 CTA 가독성을 해치지 않도록 보조층으로 유지

## 공통 상태/인터랙션 체크
- [ ] `home.expanded`: 홈 탭 + 첫 카드 확장 + 타이머 표시 상태 유지
- [ ] `home.modalOpen`: 퀘스트 추가/편집 모달 오픈 상태 관리
- [ ] 탭 전환/아코디언/체크/모달 이벤트 간 상태 충돌 없음
- [ ] 모든 인터랙션 요소 터치 타깃 `44x44px` 이상
